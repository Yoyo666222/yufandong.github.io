# Yufan Dong — Academic Website

Personal academic website for Yufan Dong, a neuroscientist studying REM sleep, cortical dynamics, cognition, and consciousness.

## Website

https://yoyo666222.github.io/yufandong.github.io/

## Content

- Research overview and current projects
- Publications with links to official articles
- Selected conference presentations
- Web CV and downloadable PDF

## Updating the site

This site uses the Academic Pages Jekyll theme and is hosted with GitHub Pages. Content is primarily stored in the _pages directory; profile and URL settings are in _config.yml, navigation is in _data/navigation.yml, and the custom visual layer is in _sass/layout/_custom-profile.scss.
